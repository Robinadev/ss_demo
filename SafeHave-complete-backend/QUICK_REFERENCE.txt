================================================================================
                        TROUBLESHOOTING QUICK FIX
================================================================================

"Cannot connect to database":
  → Check POSTGRES_PRISMA_URL in .env
  → Verify Supabase project is running
  → Run: npm run prisma:studio

"Migration failed":
  → Run: npm run db:reset
  → Then: npm run prisma:migrate

"Port 3001 already in use":
  → Kill process: lsof -ti:3001 | xargs kill
  → Or change port in .env: APP_PORT=3002

"Swagger docs not loading":
  → Check server is running on :3001
  → Verify @nestjs/swagger is installed
  → Clear browser cache and refresh

"Slow database queries":
  → Run: npm run prisma:studio
  → Check if indexes are created
  → Monitor connections: npm run logs

"JWT token expired":
  → Re-login to get new token
  → Token lifetime: 24 hours
  → Change in .env: JWT_EXPIRATION=24h
