ERROR FIXED: spawn xdg-open ENOENT
────────────────────────────────────────────────────────────────────────────

WHAT WAS THE PROBLEM?
The error "spawn xdg-open ENOENT" occurred when running npm scripts or the
application. This happens because npm packages try to open browsers using the
xdg-open command (a Linux utility) which doesn't exist in:
  • Windows environments
  • macOS (uses 'open' instead)  
  • Containerized/Sandboxed environments (Vercel)
  • CI/CD pipelines
