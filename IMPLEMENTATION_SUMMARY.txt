================================================================================
                      AUTHENTICATION FLOW
================================================================================

User Registration:
  Frontend: User fills registration form
  → Backend: Create user with hashed password
  → Database: Store user
  → Response: Confirmation message

User Login:
  Frontend: User enters email & password
  → Backend: Validate credentials
  → Response: access_token + refresh_token
  → Frontend: Store tokens in localStorage

Authenticated Requests:
  Frontend: API call with Authorization header
  → Axios Interceptor: Add "Authorization: Bearer {token}"
  → Backend: JWT Guard validates token
  → Backend: Process request if authorized
  → Response: Return data

Token Expiration:
  Frontend: API call with expired token
  → Backend: Return 401 Unauthorized
  → Axios Interceptor: Detect 401
  → Frontend: Call refresh endpoint with refresh_token
  → Backend: Validate refresh_token, return new access_token
  → Frontend: Retry original request with new token

Logout:
  Frontend: Click logout button
  → Frontend: Clear localStorage
  → Frontend: Clear AuthContext state
  → Frontend: Redirect to login page
